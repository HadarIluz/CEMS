Please follow the instructions for best use:
1. Make sure the time zone is updated in your local MySql enviroment:
	1.1 Open MySql Workbench
	1.2 Run this quary:
		set global time_zone='+2:00';

2. Run the server:
	2.1 If using JAR, just double click. If running from Eclipse, use Run Configuration of ServerUI
	2.2 Click on "Start Server" (No need to change port)
	
3. Run the Client:
	3.1 If using JAR, just double click. If running from Eclipse, use Run Configuration of ClientUI
	3.2 You can now log in
